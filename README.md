
# BIM - 实名制平台

本地测试 http://localhost:9527.

## 开发

```bash
# 克隆项目
git clone http://git.buskey.cn/hanxiang/bim-web.git

# 安装依赖
npm install

# 建议不要用cnpm安装 会有各种诡异的bug 可以通过如下操作解决 npm 下载速度慢的问题
npm install 

# 启动服务
npm run dev
```



## Build

```调试
# build for test environment
npm run dev

# 生成
npm run build:prod
```
